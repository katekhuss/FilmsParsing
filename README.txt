Проект: Films Database Loader

Проект представляет собой скрипт на PHP, который загружает информацию о фильмах из HTML-файлов в базу данных MySQL.

Описание
Скрипт сканирует каталог films и извлекает данные из HTML-файлов, включая название фильма, рейтинги на IMDb и Кинопоиске, дату выхода, страну производства, режиссера, жанр, информацию о качестве и переводе, возрастной рейтинг, продолжительность, коллекции и актеров. Полученная информация вставляется в таблицу films базы данных.

В проекте задействованы:
- Использование регулярных выражений: Для извлечения данных из HTML-файлов используются регулярные выражения.
- Работа с базой данных: Скрипт взаимодействует с базой данных MySQL для сохранения информации о фильмах.
- Многопроцессорная обработка: Обработка файлов происходит параллельно, что ускоряет процесс загрузки данных.
- Оптимизация SQL-запросов: Для вставки данных в базу используются параметризованные запросы, что обеспечивает безопасность и эффективность операций. Включена защита от SQL-инъекций.
- Централизованное подключение к базе данных: Подключение к базе данных вынесено в отдельный файл db.php, обеспечивая легкость сопровождения и повторное использование.

Использование
Убедитесь, что на вашем сервере установлен PHP и MySQL.
Создайте базу данных MySQL с именем films и импортируйте схему из файла ашдьы.sql.
Установите параметры подключения к базе данных в файле db.php.
Поместите HTML-файлы фильмов в каталог films.
Запустите скрипт index.php.

Требования
PHP >= 7.0
MySQL >= 5.6
